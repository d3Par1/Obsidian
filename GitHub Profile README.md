## 🧐 Що тут і чому

`★ Структура README — інженерія сприйняття ─────`
README рекрутер читає **вертикально, за 20 секунд**. Тому ключова інформація йде зверху → деталі внизу:

1. **Header (Hi, I'm Nazar)** — людський контакт за 1 секунду
2. **Tagline (Building toward defense tech)** — позиціонування за 2 секунди
3. **Badge buttons (LinkedIn / Telegram / Email)** — clickable CTA, recruiter може зразу пиcати
4. **TLDR** — за 10 секунд recruiter знає твій рівень, фокус, мови, готовність
5. **Currently** — показує що ти **активний**, не "сплячий" профіль
6. **Tech badges** — візуальний tech-stack замість textової стіни
7. **Projects** — короткі опуси з лінками
8. **Why defense tech** — emotional differentiator
9. **Connect** — backup contacts
10. **GitHub stats widget** — auto-updating proof of activity
