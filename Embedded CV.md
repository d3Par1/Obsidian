# ⚠️ ЧЕСНА ОЦІНКА — що ще треба, щоб CV "стрельнуло" в miltech

`★ Reality check ─────────────────────────────`
Для embedded/drone позицій CV без **жодного embedded-проєкту** буде в кінці черги. Поки ти будуєш CV вище, паралельно зроби 2–3 з цих проєктів — додай у GitHub з README і фото/відео. **Це найбільший ROI:**

1. **STM32 Blue Pill "Hello World"** — мигаючий світлодіод + UART логи в термінал. 1 вечір. ~$5 плата.
2. **ESP32 Wi-Fi telemetry** — датчик (BMP280 / IMU) → MQTT → веб-дашборд. 1 вихідний.
3. **PX4 SITL з кастомною місією** — навіть без реального дрона, в симуляторі. 0₴.
4. **MAVLink message parser в C** — read-only утиліта, що парсить `.tlog` файли. 1 вечір.

Кожен проєкт = окремий repo з README, схемою, відео-демо. **Це те, що відрізняє "студент який хоче в embedded" від "студент якого беруть на embedded".**
